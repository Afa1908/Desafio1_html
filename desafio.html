    Um pouco do surgimento do HTML

    (Voltar)

    O HTML, sigla para HyperText Markup Language, teve suas raízes nos anos 1980, quando Tim Berners-Lee, um cientista da computação britânico, estava trabalhando no CERN (Organização Europeia para Pesquisa Nuclear). Ele propôs um sistema para compartilhar informações de pesquisa entre cientistas e criou o que viria a ser o primeiro servidor web, o "CERN httpd", juntamente com o primeiro navegador web, o "WorldWideWeb" (mais tarde renomeado para Nexus), em 1990. O HTML foi desenvolvido como uma linguagem de marcação para estruturar e formatar documentos, permitindo a criação de hipertextos que podiam conter links para outras páginas. Em 1993, o HTML passou por uma padronização mais formal com a especificação HTML 2.0, o que marcou o início de sua evolução como uma linguagem fundamental para a criação de páginas da web. Desde então, o HTML passou por várias versões e evoluções, sendo atualizado e estendido para acomodar as crescentes demandas da web moderna.

    Tags utilizadas em estrutura básica no HTML

    (Voltar)

    Algumas tags básica do HTML abaixo:

    • DOCTYPE html -> Esta declaração define o tipo de documento como HTML5, que é a versão mais recente do HTML.

    • A tag <html> é a raiz do documento HTML. Todo o conteúdo da página é contido entre as tags <html> e </html> no documento HTML.

    • A tag <head> contém metadados sobre a página, como o conjunto de caracteres usado ( <head charset="UTF-8">) e o título da página (<title>Título da Página</title>). O título é exibido na barra de título do navegador.

    • A tag <body> contém o conteúdo visível da página, como texto, imagens, links, e outros elementos HTML. É aqui que você coloca o que os visitantes verão na página.

    • A tag <h1>a <h6> Tags HTML - Titulos (Cabeçalhos) Os cabeçalhos HTML são definidos com as tags de <h1> a <h6>. A tag <h1> define o cabeçalho mais importante. <h6> define o título menos importante.

    • Tags HTML - Parágrafos Os parágrafos HTML são definidos com a tag <p>:

      Isto é um parágrafo usando a tag <p>.

      Isto é um outro parágrafo usando a tag <p>.

    • O HTML contém vários elementos para definir o texto com um significado especial. Os elementos de formatação foram projetados para exibir tipos especiais de texto:

      • <b>- Texto em negrito
      • <strong>- Texto importante
      • <i> - Itálico
      • <em>- Texto enfatizado
      • <mark>- Texto marcado
      • <small> - Texto menor
      • <del>-Texto deletado
      • <ins>- Texto inserido
      • <sub> - Texto subscrito
      • <sup> - Texto sobrescrito

    • Tag <a> é usada para adicionar Links como (Ex: < a href="https://www.exemplo.com"> Link para o Exemplo </a>)

      Outro uso para a tag de âncora </a> é a demarcação de destinos para links dentro do próprio documento.

    Utilizar novas tags sugeridas no HTML

    (Voltar)

    O HTML5 introduziu várias novas tags que podem ser usadas para melhorar a estrutura e a semântica de suas páginas web. Aqui estão algumas das tags mais úteis e suas finalidades no HTML abaixo:

    • A tag <header> é usado para representar o cabeçalho de uma seção ou de todo o documento. Pode conter logotipos, títulos, menus de navegação, etc.

    • A tag <nav> é utilizado para definir uma seção de navegação, geralmente contendo links para outras partes do site..

    • A tag <article> é usado para identificar conteúdo independente e autocontido, como um post de blog, notícia ou artigo.

    • A tag <section> serve para agrupar conteúdo relacionado dentro de um documento. Pode ser usado para dividir um artigo em várias seções.

    • A tag <aside> é usado para conteúdo relacionado, mas não essencial, como barras laterais, widgets, anúncios, entre outros.

    • A tag <main> define o conteúdo principal da página. Deve haver apenas um elemento ` <main> por página.

    • A tag <footer> representa o rodapé de uma seção ou do documento inteiro, geralmente contendo informações como direitos autorais, informações de contato, etc..

      • Lembre-se de que a implementação de tags HTML5 pode variar entre os navegadores, por isso é importante testar sua página em diferentes navegadores para garantir que as novas tags funcionem corretamente. Além disso, usar tags HTML5 apropriadas ajuda na acessibilidade, SEO e na interpretação correta do conteúdo por parte dos motores de busca. Portanto, é recomendável usá-las sempre que apropriado..
